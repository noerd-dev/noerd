# Example Application

The best way to learn Noerd is by exploring the **Noerd Example Study App**f. It demonstrates all core concepts — modules, lists, details, relations, tabs, translations, and tests — in a complete, working application.

- **GitHub Repository:** [github.com/noerd-dev/noerd-example](https://github.com/noerd-dev/noerd-example)
- **Live Demo:** [demo.noerd.dev](https://demo.noerd.dev)

Clone the repository and follow the README to get started.
